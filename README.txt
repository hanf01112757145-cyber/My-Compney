طريقة التشغيل على XAMPP:
1) فك الضغط.
2) انسخ فولدر hiresmart بالكامل داخل:
   C:\xampp\htdocs\
3) شغّل Apache و MySQL من XAMPP.
4) افتح:
   http://localhost/phpmyadmin
5) اعمل Database باسم:
   hiresmart
6) ادخل على قاعدة البيانات hiresmart ثم Import.
7) اختار ملف قاعدة البيانات:
   C:\xampp\htdocs\hiresmart\database\hiresmart.sql
8) اضغط Go.
9) افتح المشروع:
   http://localhost/hiresmart/
